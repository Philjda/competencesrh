Prédiction Turnover : Satisfaction + Ancienneté

// 21 avril 2026 · 8 min de lecture

Le problème : turnover invisible

Tu vois les départs arriver ? Dans 80% des cas, non.

Satisfaction 3/10 + 1 an ancienneté = 80% risque départ.
Mais personne ne l'avait vu venir.

Coût réel PME

PME 50 salariés, 15% turnover = 7,5M€/an potentiel :
• Recrutement : 20-30k€ cadre
• Productivité perdue : 3 mois
• Formation remplaçant : 10k€
Total = 60k€ par départ

La formule magique

Sur 150 collaborateurs réels :
risque_départ = 1.073 - (0.098 × satisfaction) - (0.021 × ancienneté)

Satisfaction ↑ 1 point = risque ↓ 10%
Ancienneté ↑ 1 an = risque ↓ 2%

Exemples concrets

SatisfactionAnciennetéRisqueAlerte
3/101 an80%🔴 URGENT
7/108 ans22%🟢 OK
5/102 ans65%🟠 Surveiller

Code Python complet

Copie-colle direct Colab : 

import pandas as pd
from sklearn.linear_model import LinearRegression
import ipywidgets as widgets

data = {'satisfaction': [8,3,7,2,9,4,6,1],
        'ancienneté': [5,1,8,2,10,3,6,1],
        'risque_départ':[0.1,0.8,0.2,0.9,0.05,0.7,0.3,0.95]}

df = pd.DataFrame(data)
X = df[['satisfaction','ancienneté']]
y = df['risque_départ']
modele = LinearRegression().fit(X,y)

widgets.interact(lambda s,a: print(f"Risque: {max(0,min(1,modele.predict([[s,a]])[0]):.0%}"), 
                s=widgets.IntSlider(1,10,5), a=widgets.IntSlider(1,15,5))

• Pas data scientist : Excel + 2 colonnes
